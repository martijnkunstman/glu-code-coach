Installeer de Coding Coach in VS Code

Een coach in je editor die je begeleidt bij programmeeropdrachten — met hints en vragen, geen kant-en-klare antwoorden. Installeren duurt een halve minuut.

Stap 0 · Download

coding-coach.vsix · werkt op Mac & Windows
⬇ Download de extensie

Daarna in 4 stappen

  1. Open het Extensions-paneel in VS Code
    Klik links op het blokjes-icoon, of druk Cmd+Shift+X (Mac) / Ctrl+Shift+X (Windows).
  2. Kies “Install from VSIX…”
    Klik rechtsboven in het paneel op de ··· en kies Install from VSIX…. Selecteer het gedownloade coding-coach.vsix.
  3. Wacht op “Completed installing”
    Verschijnt er een knop Reload? Klik die even aan.
  4. Open de coach en begin
    Klik op het 🎓 Coding Coach-icoon links, kies bovenin een opdracht en ga aan de slag. Je hoeft niets in te stellen.

Lukt het niet? Vraag het je docent. · Je hebt geen account of sleutel nodig.
